# Student Self-Government System

## Overview

Student Self-Government System is a client-server information system developed as a bachelor's graduation project.

The system is designed to support student self-government activities, automate event management, handle student requests, and maintain activity records.

## Architecture

The project consists of three main components:

### Backend

* Python
* FastAPI
* PostgreSQL
* REST API

### Telegram Bot

* Python
* Aiogram

### Desktop Application

* C#
* Windows Forms

## Main Features

### Student Functions

* View news and announcements
* Browse upcoming events
* Register for events
* Submit requests and appeals
* View personal activity points

### Administrative Functions

* Manage news
* Manage events
* Process student requests
* Award activity points
* Manage user accounts
* View system logs
